======================
Resource indirect URIs
======================


An URI that identifies a resource by the RDF graph it contains.

.. raw:: html

	<form action="http://fenfire.org/2005/resource" method="GET">
	<textarea name="description" rows=20 cols=80>
	@prefix foaf: <http://xmlns.com/foaf/0.1/>.

	&lt;> foaf:primaryTopic [
	    foaf:homepage <http://fenfire.org/>
	].
	</textarea>
	<input type="reset">
	<input type="submit">
	</form>
